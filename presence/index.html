Layer 1

Canonical corpus first

Extract, deduplicate, chunk, and store agent sessions as steerable markdown in ~/.aicx/. This is the ground truth.

Layer 2

Semantic memex second

Materialize the canonical corpus into vector + BM25 retrieval for agents and MCP tools without giving up the file-backed source of truth.

Operator UX

Built for humans who want one obvious next move

aicx, aicx doctor, aicx dashboard --open, aicx latest, aicx read, guided install paths, and background memex upkeep make the capability surface stronger and easier at the same time.

Why it exists

Capability surface and user friendliness do not need to fight.

AI Contexters is opinionated on purpose: keep canonical truth visible, keep semantic retrieval optional but close at hand, and give the operator a surface that explains itself instead of forcing them to memorize internals.

Three-step flow

From raw logs to useful recall

01

Install once

Use crates.io, or extract a release archive and run ./install.sh if you do not want a Rust toolchain.

02

Run aicx

Let the guided front door tell you the current verdict, the next recommended move, and when it makes sense to open the deeper aicx doctor report or use --fix.

03

Open the browser or go straight to recall

Use aicx dashboard --open when you want a calmer browser front door, or start with aicx latest when you want the newest readable chunks right away.
